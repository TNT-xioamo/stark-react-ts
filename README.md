# stark-clone
> 这是一个极简的 react 项目，引入了乾坤作为微前端解决方案

## Build Setup
```bash
# 克隆项目
git clone 

# 进入项目目录
cd stark-clone

# 安装依赖
yarn

# 建议不要直接使用 cnpm 安装以来，会有各种诡异的 bug。可以通过如下操作解决 npm 下载速度慢的问题


# 启动服务
yarn start
```
浏览器访问 [http://localhost:3000](http://localhost:3000)

## 发布

```bash
# 构建测试环境
yarn test

# 构建生产环境
yarn build
```
